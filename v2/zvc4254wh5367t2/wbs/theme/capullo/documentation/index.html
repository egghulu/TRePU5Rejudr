Introduction

Capullo - is a responsive coming soon theme built using stable Bootstrap v3.3.6 and HTML5/CSS3 features and suitable for creative agencies and any kind of freelancers, photographers or small business which need a fast way to showcase their work.

Capullo - is a fully responsive, it works nicely on smartphones, tablet PCs and desktops.

Requirements

Capullo is an HTML Template. To customize this template, You should know some basic HTML. If you don't have this knowledge, you can contact any Web Designer to do this task. Editing this template with out knowing things means, it will break the layout and design.

Fonts

You can change a default list of google fonts to a file style.css. For example :

  <link href='http://fonts.googleapis.com/css?family=Open+Sans:300,600' rel='stylesheet' type='text/css'>

Changing Logo

To change the current logo to your Logo, Create it separately in any Graphic software and Export it in PNG format, so that the backgground will be transparent. :

  <img alt="" class="logo" src="images/logo-big.png">

  <img alt="" src="images/logo-small.png">

Email Subscription

You need a mailchimp account to get the form working. You will get an URL, API code and List ID from Mailchimp dashboard. Copy it in your html file:

  <form class="form-inline" id="mailchimp-form" action="http://capullo.us9.list-manage.com/subscribe/post?u=a8cd4fa72585379e205821fa2&id=366e69e674">

Background Images

To change the current background, you can use any file image such as svg, jpg, png. The size is 1600x1067 pixels for hero image and 1024x683 pixels for footer image. You can keep the same name, so that you don't need to edit CSS:

  <section id="hero" class="hero hero-countdown background-img" style="background-image:url('images/hero-background.jpg');">

  <section id="social" class="padding-y-lg background-img" style="background-image:url('images/footer-background.jpg')">
